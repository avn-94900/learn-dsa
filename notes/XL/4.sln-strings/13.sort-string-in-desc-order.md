## 8. Sort a String in Descending Order

### âœ… Problem

Given a string, sort its characters in **descending order (Z â†’ A)**.
Example:
Input: `"programming"`
Output: `"rrponmmigg"`

### ðŸ“˜ Approach

* Convert string â†’ char array.
* Use `Arrays.sort()` with custom comparator (descending).
* Convert back to string.

### â± Complexity

* Sorting: **O(n log n)**
* Space: **O(n)**

### ðŸ’» Code

```java
import java.util.Arrays;
import java.util.Collections;

public class SortStringDescending {
    public static String sortDescending(String input) {
        // Convert to Character[] (not char[]) for custom comparator
        Character[] chars = input.chars()
                                 .mapToObj(c -> (char) c)
                                 .toArray(Character[]::new);
        /*
        Character[] chars = new Character[input.length()];
        for (int i = 0; i < input.length(); i++) {
            chars[i] = input.charAt(i);
        }
        */

        // Sort in reverse order
        Arrays.sort(chars, Collections.reverseOrder());

        // Build result
        StringBuilder sb = new StringBuilder(chars.length);
        for (char c : chars) {
            sb.append(c);
        }
        return sb.toString();
    }

    public static void main(String[] args) {
        String input = "programming";
        System.out.println("Descending order: " + sortDescending(input));
    }
}
```
