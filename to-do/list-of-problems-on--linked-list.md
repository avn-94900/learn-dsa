## Common Problem Variations by Difficulty

### Easy Level
- Basic operations, traversal, simple reversal, merge two sorted lists

### Medium Level  
- Reverse in K groups, detect cycle, find intersection, remove duplicates, add numbers

### Hard Level
- Clone with random pointer, LRU cache, merge K sorted lists, complex rearrangements

## Key Techniques to Master
1. **Two Pointer Technique** - Fast/slow pointers
2. **Dummy Node Usage** - Simplify edge cases
3. **Recursive vs Iterative** approaches
4. **In-place operations** - Space optimization
5. **Edge case handling** - Empty list, single node, etc.
