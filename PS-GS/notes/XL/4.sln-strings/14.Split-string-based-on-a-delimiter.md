## Split String Based on a Delimiter

Split a string using a delimiter.
Example: `"apple,banana,grape"`, delimiter `,` â†’ `[apple, banana, grape]`

### ğŸ“˜ Approach

* Use `String.split(regex)`.

### â± Complexity

* **O(n)**

### ğŸ’» Code

```java
import java.util.Arrays;
import java.util.ArrayList;
import java.util.List;

public class StringSplitter {

    // Method 1: Using String.split()
    public static String[] splitByDelimiter(String input, String delimiter) {
        return input.split(delimiter);
    }

    // Method 2: Manual split using char delimiter
    public static String[] splitByCharDelimiter(String input, char delimiter) {
        List<String> result = new ArrayList<>();
        int start = 0;
        int index;

        while ((index = input.indexOf(delimiter, start)) != -1) {
            result.add(input.substring(start, index));
            start = index + 1;
        }
        result.add(input.substring(start)); // last token
        return result.toArray(new String[0]);
    }

    public static void main(String[] args) {
        String input = "apple,banana,grape";

        // Method 1
        String[] parts1 = splitByDelimiter(input, ",");
        System.out.println("Method 1: " + Arrays.toString(parts1));

        // Method 2
        String[] parts2 = splitByCharDelimiter(input, ',');
        System.out.println("Method 2: " + Arrays.toString(parts2));
    }
}
```
